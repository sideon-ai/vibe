## 12.2.2 프로젝트 계획 세우기
~~~
스터디 모임 매칭 플랫폼 '윗미(Wit.me)'
사용자:
•	일반 사용자: 스터디 검색 및 참여 신청
•	모임장: 스터디 생성(최대 5개), 신청자 승인/거절, 참가자 명단 다운로드
기술:
•	Gemini API (gemini-2.0-flash 또는 gemini-2.5-flash 모델 사용)
•	Firebase Authentication (이메일/비밀번호 인증)
•	Firebase Firestore (데이터 저장)
요구사항:
•	스터디 카드형 목록 보기
•	상세 정보 및 참여 신청 기능
•	모임장 전용 신청자 관리 페이지
•	CSV 형식 참가자 명단 다운로드

[API 키]

파이어베이스 키: (직접 입력)
